A) Installing the plugin - top

Installing the plugin is very easy just like any other wordpress plugin.
Connect to your hosting and upload the extracted zip file folder of the plugin into wp-content/plugins/
Then go to wp-admin->plugins->and activate WP CaptchaCO.IN

B) Setting Plugin Options - top

The first thing you might want to do is to setup WP CaptchaCO.IN Options :
Into wordpress admin, go to the newly installed plugin section called "WP CaptchaCO.IN".

  • Provide your correct Application Key which can be found in your https://www.captchaco.in dashboard.

C) Adding Shortcode - top

Adding the shortcode is a simple straight forward job. You can use it on as many pages/posts as you want.
The shortcode contains a start tag [wp_captchacoin_popup captcha_id=THE_CAPTCHA_ID] and an end tag [/wp_captchacoin_popup].
The content between the shortcode will not be shown unless they pay the corresponding bitcoin amount.

Example: [wp_captchacoin_popup captcha_id=THE_CAPTCHA_ID] this content will be shown to paying visitors [/wp_captchacoin_popup]

Get the shortcode in the "Statistics & Shortcodes" page.
